# Outputs

All files generated by running `python main.py`. Regenerated on every run.

> **Note:** Development commands and run notes were tracked in Notion during this project.

## Files (13)

---

### HTML Report (1)

**`report.html`**
The main deliverable. Open this in any browser to see the full interactive report.

- All 4 scenarios shown in separate cards
- Each scenario has tabs: **Live 2D/3D Canvas**, **Path Plot**, **Distance Chart**, **Summary**
- Live canvas: animated drone flight you can play/pause, change speed, scrub through time
- 3D scenarios (3A, 3B) have an isometric 3D view — drag to rotate, altitude shadows visible
- Conflict markers pulse red; safe scenarios show green status
- Fully self-contained — no internet needed, works offline
